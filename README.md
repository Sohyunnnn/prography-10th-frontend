## 💻 프로젝트 실행 방법
### 1️⃣ 프로젝트 클론
```
git clone https://github.com/Sohyunnnn/prography-10th-frontend.git
```

### 2️⃣ 패키지 설치
```
npm install
```
### 3️⃣ 개발 서버 실행
```
npm run dev
```

### 배포링크 
[🚀 프로그래피 10기 프론트엔드 과제](https://prography-10th-frontend-phi.vercel.app/)
